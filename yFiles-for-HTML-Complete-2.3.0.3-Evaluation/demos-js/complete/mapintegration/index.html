Description

This demo shows how to integrate a GraphComponent with Leaflet. The GraphComponent is included in a custom Leaflet-Layer. The placement of the nodes uses geo-coordinates to get the correct locations on the map.

Things to Try

  • Drag the map and see how the graph moves along.
  • Zoom into the map using the mouse-wheel or the controls in the upper-left corner of the map. The nodes mark the location of the airports and will keep their geo-location. Some of them are only visible on larger zoom levels.
  • Click on two nodes and see the shortest path between those nodes.
  • Toggle the graph mode using the -control in the upper-right corner of the map to free the nodes from their geo-locations and apply a radial layout that visualizes how many hops are needed to reach another airport. Change the center node with a double-click on another node.
