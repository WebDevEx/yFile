Source Code Demos and Tutorials

Extensive source code demo applications and tutorials are located in the demos-js and demos-ts directories of the yFiles for HTML package. All demos are available as JavaScript. For many of these demos, the TypeScript sources are also available in the demos-ts directory.

These demos are also hosted at live.yworks.com and their source code is available at the yfiles-for-html-demos GitHub project. Note that you're not allowed to upload the yFiles for HTML redistributables to GitHub or any other file sharing site.

Running the Demos

Due to the use of ECMAScript import statements with symbolic names, most demos do not work when launched from the file system or served through a web server with default configuration. Therefore, we recommended to access the demos through the included demo web server.

The demo server requires Node.js to be installed. To install Node.js, go to nodejs.org, download the installer that is suitable for your system, and install it. Both the 'LTS' and the 'Current' version will work fine.

Then, you can start the demo server either with the corresponding run configuration in the WebStorm project, with the corresponding task in the Visual Studio code workspace, by double-clicking the start_demos.cmd script, or by running the following command in a terminal: 

> npm start

When the server is up and running, its welcome page should open in your browser. If it doesn't, please direct your browser to http://localhost:4242/README.html.

Running the TypeScript Demos

The TypeScript demo sources in demos-ts are compiled as part of the initial installation script. The TypeScript demos can also be browsed using the demo server from http://localhost:4242/demos-ts/README.html.

If you want to edit the TypeScript sources, you can use the watch task to instantly view the results: 

> cd demos-ts
> npm run watch

API Documentation and Developer's Guide

The Documentation Viewer is located in the directory doc/api. It can be used to browse and search both the API documentation and the Developer's Guide.

The introduction videos page lists tutorials and screencasts about working and developing applications with yFiles for HTML.

The Developer's Guide describes in detail the architecture and all main features of yFiles for HTML.
